# ASP.NetCore

## DotNetCore

- .NetCore
  - CoreFx:基础类库
  - CoreCLR:含RyuJIT的运行平台
  - .Net Compiler Platform：编译器平台Roslyn
  - CoreRT:采用AOT的运行时
  - LLILC:跨平台的MSIL编译器
  - docfx:构建技术文档的平台

- dotnet restore
  - dotnet restore [root/project.json]

- dotnet run
